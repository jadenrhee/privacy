TrustLens Privacy Policy

_Last updated: July 7, 2026. Applies to TrustLens 1.0.0._

Plain English, one screen. This policy matches what the code actually does; it

was verified line-by-line against the extension during a QA audit.

What TrustLens collects about you

Nothing. No account, no analytics, no telemetry, no browsing history, no

product views, no crash reporting. There is no server that receives data about

you, so there is nothing for us to see, sell, or lose.

What happens on your device

When you open an Amazon product page, TrustLens analyzes the content already on

that page (reviews, ratings, dates, prices) inside your browser, and stores a few

things locally so the extension works:

  • Your settings (display mode, per-site pauses).
  • Price memory: prices of products you have viewed, so it can show "you saw

this cheaper last month." Stored only on your device, capped in size, never

uploaded.

  • A cached copy of the scoring configuration (see below).
  • A short local error log to help you file a bug report if something breaks.

On your device only, never transmitted.

Uninstalling the extension deletes all of it.

The only network request the extension makes

TrustLens periodically downloads its own scoring-configuration file — a small

JSON file of weights and page selectors — so scoring can be tuned and kept

working when Amazon changes its layout, without waiting for a store update. This

request contains no information about you beyond what any file download reveals

(your IP address, to the file's host). We do not log it. If the download fails,

the extension falls back to the configuration bundled inside it and keeps working.

Permissions, and why each is needed

  • Access to www.amazon.com — to read the product page you are viewing and

show the trust overlay there. TrustLens runs on no other site.

  • Storage — for the local settings, price memory, and cached config above.

Nothing is synced or transmitted.

  • Alarms — to schedule the daily configuration refresh.

If you uninstall

Chrome opens a static goodbye page from the project site. It carries no

parameters, no identifier, and no tracking; it exists only so you can send

feedback if you want to. The uninstall itself deletes everything TrustLens

stored.

Payments

TrustLens 1.0.0 is entirely free and contains no payment processing. If a paid

"Pro" tier is added in a future version, this policy will be updated first to

describe exactly what the payment processor receives (at most your payment status

and the email you give them, used only to manage your subscription), and the

change will be announced in the extension before it takes effect.

Changes

Material changes will be announced in the extension before they apply, and the

history is public in the project repository.
